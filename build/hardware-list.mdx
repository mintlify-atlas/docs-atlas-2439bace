The Voron Tower replaces the standard 2020 aluminum extrusion frame with 4040 series extrusions. The majority of hardware — fasteners, motion components, electronics, and the bed — follows the standard Voron 2.4 R2 BOM. Only the frame extrusions and Z-axis rails differ significantly.

<Note>
  The official hardware list spreadsheet for the 350×350mm build is available on GitHub: [Hardware List 350×350mm.xlsx](https://github.com/PorcoMaster/VoronTower/blob/main/Voron%20Tower/Hardware%20List%20350x350mm.xlsx). Download it for the authoritative quantities and part numbers.
</Note>

## Frame extrusions

All frame members use **4040 series 10 aluminum extrusion** (slot size 10mm, profile 40×40mm). This is a direct upgrade from the 2020 profile used in stock Voron 2.4.

<Info>
  Gantry extrusions should be cut to lengths ending in **0 or 5 mm** to align with the parametric model. Safe adjustment range is ±3mm for `gantry_x_front`, `gantry_x_back`, and `gantry_y` from the calculated values.
</Info>

| Member | Qty | Notes |
|---|---|---|
| Vertical corner extrusions | 4 | Height depends on Z travel + offsets |
| Bottom horizontal extrusions (X) | 2 | 350mm build width + offsets |
| Bottom horizontal extrusions (Y) | 2 | 350mm build depth + offsets |
| Top horizontal extrusions (X) | 2 | Same as bottom |
| Top horizontal extrusions (Y) | 2 | Same as bottom |
| Gantry X front | 1 | Cut to round number per parameter table |
| Gantry X back | 1 | Cut to round number per parameter table |
| Gantry Y (left & right) | 2 | Cut to round number per parameter table |

## Linear rails

The Z axis is upgraded from MGN9 to MGN12 for increased load capacity and rigidity. XY motion retains standard Voron 2.4 R2 rail specifications.

| Axis | Rail type | Qty | Notes |

## Fasteners

Fastener requirements are largely identical to the Voron 2.4 R2 BOM. The 4040 extrusion uses M5 T-nuts (roll-in or hammer) with the 10mm slot profile — confirm T-nut compatibility with your extrusion supplier.

| Type | Sizes | Notes |
|---|---|---|
| Socket head cap screws | M3×6, M3×8, M3×10, M3×12, M3×16, M3×20, M3×25, M3×30 | Standard Voron 2.4 R2 quantities |
| Socket head cap screws | M5×10, M5×12, M5×16 | Frame and rail mounting |
| Button head screws | M3×6, M3×8 | Panel and door clips |
| T-nuts (roll-in) | M3, M5 | Must be 10mm slot profile for 4040 extrusion |
| Heat-set inserts | M3×5×4 | Standard Voron 2.4 R2 quantities |
| Hex nuts | M3, M5 | Standard Voron 2.4 R2 quantities |

<Warning>
  Standard 2020 M5 T-nuts (6mm slot) are **not compatible** with 4040 series 10 extrusion (10mm slot). Verify the slot width of your 4040 extrusion purchase and order matching T-nuts.
</Warning>

## Electronics

All electronics are unchanged from the stock Voron 2.4 R2. Use the official Voron BOM for:

- Controller board (e.g., Octopus, Manta M8P)
- Raspberry Pi or equivalent SBC
- Stepper drivers (TMC2209 recommended)
- Power supplies (24V bed PSU + 24V main PSU)
- Wiring, connectors, and cable management
