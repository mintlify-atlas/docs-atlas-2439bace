## Before you begin

Voron Tower is a modification of the **Voron 2.4 R2** — not a standalone printer design. You need a working knowledge of the stock Voron 2.4 R2 build before attempting this mod.

<Info>
  Everything not listed in the modified parts table reuses standard Voron 2.4 R2 hardware and printed parts. Build the stock machine first if you haven't already.
</Info>

<Steps>

<Step title="Get the files">

The STL files and individual Fusion 360 source files are on GitHub. The full parametric assembly file is too large for GitHub and is hosted separately on Google Drive.

<CardGroup cols={2}>
  <Card title="GitHub repository" icon="github" href="https://github.com/PorcoMaster/VoronTower">
    STL files, individual Fusion 360 source files, hardware list, and documentation.
  </Card>
  <Card title="Main Fusion 360 assembly" icon="cube" href="https://drive.google.com/file/d/1ePI7Y9nos1gBQOqrRFIDSbPcuf1UqJJw/view?usp=sharing">
    The complete parametric assembly. Required if you want to adjust build volume or frame dimensions.
  </Card>
</CardGroup>

<Note>
  The STL files in the GitHub repo are pre-exported for a **350×350mm** build. If you want a different size, open the main Fusion 360 assembly, adjust the parameters, and re-export the affected parts.
</Note>

</Step>

<Step title="Choose your build size">

The recommended build size is **350×350mm**. A 500×500mm build is possible but less tested — the gantry still uses a 2020 carriage at that size, which limits confidence in the larger variant.

<Warning>
  Do not change gantry parameters by more than ±3 mm from the defaults. The gantry profiles have a 6 mm tolerance window built in; exceeding it will cause fit issues. Use `gantry_x_front`, `gantry_x_back`, or `gantry_y` if adjustments are needed.
</Warning>

</Step>

<Step title="Review the hardware list">

A complete bill of materials for a 350×350mm build is included in the repository at `Voron Tower/Hardware List 350x350mm.xlsx`. Open it before ordering anything — it covers all 4040 extrusion lengths, fasteners, and motion hardware specific to this modification.

<Tip>
  The gantry profile cut lengths in the current Fusion 360 file have been standardized to round numbers ending in 0 or 5 mm, making them easier to order from extrusion suppliers.
</Tip>

</Step>

<Step title="Understand the Z axis change">

One of the most significant mechanical changes in Voron Tower is the Z axis linear rail upgrade. The stock Voron 2.4 R2 uses **MGN9** rails on Z. Voron Tower uses **MGN12** rails.

When ordering your hardware, make sure to source MGN12 carriages and rails at the correct length for your build size — MGN9 hardware from a stock Voron 2.4 BOM will not work here.

</Step>

## Tools you need

<CardGroup cols={2}>
  <Card title="3D printer" icon="print">
    Any printer capable of printing ABS or ASA. The parts are sized for a Voron 2.4 toolhead but can be printed on most enclosed printers.
  </Card>
  <Card title="Standard Voron tools" icon="screwdriver-wrench">
    The same hand tools used for a stock Voron 2.4 build: hex keys, soldering iron for heat-set inserts, flush cutters.
  </Card>
  <Card title="Fusion 360 (optional)" icon="cube">
    Only required if you want to adjust parameters and re-export STLs for a custom build size. The pre-exported STLs cover a standard 350×350mm build.
  </Card>
  <Card title="Spreadsheet software" icon="table">
    To open `Hardware List 350x350mm.xlsx` and track your parts order.
  </Card>
</CardGroup>
