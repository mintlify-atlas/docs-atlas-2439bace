## What is Voron Tower?

Voron Tower is a community-driven modification of the [Voron 2.4 R2](https://vorondesign.com/) 3D printer that replaces the standard 2020 aluminum extrusion frame with a beefy **4040 series extrusion** profile. The result is a more rigid, stable machine capable of larger build volumes — designed to be built as a 350×350mm or scaled up to 500×500mm.

The design is **fully parametric** in Fusion 360, meaning you can adjust the build volume, frame dimensions, gantry offsets, and more — all from a central parameter table — without manually editing individual sketches.

</Note>

## What's modified from stock Voron 2.4?

Voron Tower is not a from-scratch design — it builds on the proven Voron 2.4 R2 platform. The following parts have been redesigned for 4040 extrusions:

| Component | Change |
|---|---|
| Z Belt Drive | Redesigned for 4040 frame mounting |
| Z Idler | Adapted for 4040 rail position |
| Z Bearing Block | Top and bottom merged into one unified block |
| Front Idler (Top & Bottom) | New geometry for 4040 profile |
| A/B Drive | Updated mounting for 4040 gantry |
| Z Chain Lower Mount | Repositioned for new frame |
| Panel Clips | Corner and midspan clips for 4040 extrusions |
| Door Clips | Handle, hinge, and magnetic latch for 4040 doors |

Everything else — the toolhead, electronics, bed, gantry motion system — reuses standard Voron 2.4 R2 parts.

## Project status

<Info>
  This project is actively developed. The recommended starting build size is **350×350mm**. The 500×500mm variant is possible but less tested, as the 2020 carriage is retained on the larger build.
</Info>

The Fusion 360 master file is fully parametric. Recent updates have:
- Fixed all frame joint types and added joint animations
- Corrected build volume calculations and added configurable offsets
- Standardized gantry cut lengths to round numbers (ending in 0 or 5 mm)
- Added `full_height`, `full_width`, `full_depth` parameters for total printer dimensions
